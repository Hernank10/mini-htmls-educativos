Suite pedagógica: Análisis financiero aplicado

Calculadoras interactivas (ROE, P/E, Valor intrínseco y costo de combustible) adaptadas a: finca raíz, vehículos usados, entrega en moto y productos agrícolas.

ROE y P/E

Calcula rentabilidad sobre el capital (ROE) y relación Precio/Ganancia (P/E). Ajusta ganancias por costos (combustible, mantenimiento).

Valor Intrínseco (fórmula simple)

Fórmula: (Ingresos o beneficio × (1 + g)) / tasa_descuento. Útil para estimar valor de negocio/activo.

Ejercicio avanzado: crecimiento por etapas

Puedes usar el modo avanzado para proyectar n años con g1 y luego crecimiento perpetuo g2. (Incluido en la documentación).

Costo del Combustible (Gasolina / ACPM)

Calcula el gasto en combustible por viaje o por período. Útil para restar a ganancias.

Ejercicios y ejemplos listos

Carga ejemplos preconfigurados para cada sector o descarga los datos en CSV para experimentar en Python o Excel.

Instrucciones rápidas
  1. Elige el sector y ajusta las cifras (ganancias, inversiones, consumos).
  2. Calcula ROE/P-E para ver rentabilidad y "años de recuperación".
  3. Calcula valor intrínseco y compáralo con el precio/valor del activo.
  4. Incluye el costo de combustible para obtener ganancias netas reales.
