Reglas: uso de i, y y dígrafos
  • La i es vocal (tónica o átona); aparece en diptongos y triptongos.
  • La y funciona como conjunción y como vocal/semiconsonante en posición final.
  • Dígrafos: ll, ch, qu, gu, rr.

Pide al usuario que vuelva a escribir estas reglas para memorizarlas.

Código: verificar_ll_y.py
import re

def verificar_ll_y(palabra):
    resultado = {}
    if 'll' in palabra:
        resultado['ll'] = "La palabra contiene el dígrafo 'll'. Ejemplos: 'llama', 'calle'."
    if 'y' in palabra:
        resultado['y'] = "La palabra contiene la letra 'y'. Ejemplos: 'rey', 'hoy'."
    if not resultado:
        resultado['ninguno'] = "La palabra no contiene 'll' ni 'y'."
    return resultado

if __name__ == '__main__':
    while True:
        palabra = input('Introduce una palabra (salir para terminar): ').strip().lower()
        if palabra == 'salir':
            break
        print(verificar_ll_y(palabra))

def mostrar_reglas():
    reglas = """
1. Uso de la 'i':
   - Vocal tónica o átona.
   - Presente en diptongos y triptongos.

2. Uso de la 'y':
   - Conjunción copulativa.
   - Fonema en posición final.

3. Dígrafos:
   - ll, ch, qu, gu, rr
"""
    print(reglas)

if __name__ == '__main__':
    mostrar_reglas()
    input('\nEscribe las reglas que has leído y presiona Enter:\n')
    print('Gracias. Sigue practicando.')
Código: juegos_palabras.py
import random

palabras = ['sol','luna','estrella','nube','viento','mar','flor','árbol','río']

def generar_frase(palabras):
    frase = ' '.join(random.sample(palabras, min(len(palabras),5)))
    return frase.capitalize() + '.'

if __name__ == '__main__':
    print('Frase:', generar_frase(palabras))
