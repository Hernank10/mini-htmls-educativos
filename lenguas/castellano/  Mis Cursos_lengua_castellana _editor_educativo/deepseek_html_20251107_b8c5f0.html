Domina la Gramática Española con Tecnología

Una suite completa de programas educativos diseñados para mejorar tu dominio del español mediante práctica interactiva, ejercicios personalizados y retroalimentación inmediata.

Características Principales

Aprendizaje Activo

Sistema de retroalimentación inmediata basado en principios pedagógicos modernos que fomenta el aprendizaje mediante la práctica y corrección de errores.

Gamificación

Sistema de puntuación, temporizador y logros que transforma el aprendizaje en una experiencia motivadora y entretenida.

Soporte de Audio

Integración con sistemas de texto a voz (TTS) para reforzar el aprendizaje multimodal y mejorar la pronunciación.

Exportación a PDF

Genera informes detallados de tu progreso y ejercicios completados para compartir con tutores o mantener un registro.

Ejercicios Personalizados

Crea y guarda hasta 100 ejercicios adaptados a tus necesidades específicas y áreas de mejora.

Multiplataforma

Disponible para Windows, macOS, Linux, Android y Termux, permitiendo practicar en cualquier dispositivo.

Tecnologías Implementadas

Lenguajes de Programación

Python C++ Kotlin Java

Cada implementación aprovecha las fortalezas de su lenguaje para ofrecer una experiencia óptima:

  • Python - Versión Base

    Prototipo inicial con interfaz de consola, ideal para aprendizaje de fundamentos.

  • C++ - Alto Rendimiento

    Interfaz gráfica con Raylib, optimizada para dispositivos con recursos limitados.

  • Kotlin - Android Moderno

    Aplicación nativa para Android con Jetpack Compose y material design.

  • Java - Multiplataforma

    Aplicación desktop con JavaFX, exportable a múltiples sistemas operativos.

Librerías y Frameworks

Interfaz Gráfica
  • Raylib (C++)
  • JavaFX (Java)
  • Jetpack Compose (Kotlin)
Funcionalidades Avanzadas
  • FreeTTS (Síntesis de voz)
  • Apache PDFBox (Exportación)
  • JSON (Persistencia de datos)
Ejemplo de Código - Validación Gramatical
// Función para validar modos verbales en español
function validarModoVerbal(verbo1, verbo2, oracion) {
  if (verbo1.requiereIndicativo && verbo2.requiereSubjuntivo) {
    return "No se pueden fusionar: use estructuras separadas";
  }
  return "Estructura válida";
}

Compatibilidad Multiplataforma

macOS

Compatible con versiones desde macOS Mojave (2018) hasta las más recientes, incluyendo soporte para chips M1/M2.

Requisitos: Python 3.7+ o JDK 17+
Windows

Funciona en Windows 10 y 11, con opciones de instalación portable y con instalador gráfico.

Requisitos: .NET Framework 4.8 o superior
Linux

Compatible con distribuciones basadas en Debian/Ubuntu y Red Hat/Fedora, mediante paquetes nativos.

Requisitos: GLIBC 2.28+
Android

Versión nativa para Android 8.0+ y aplicación terminal para Termux con soporte gráfico opcional.

Requisitos: 2GB RAM, Android 8.0+

Descargas e Instalación

Instrucciones de Instalación

Recomendado para principiantes y uso educativo:

# Instalar dependencias
pip install colorama pyttsx3 pdfkit

# Ejecutar programa
python3 practicador_gramatical.py
Descargar Python Version

Para máximo rendimiento y sistemas con recursos limitados:

# Compilar (Linux/macOS)
g++ -o practicador main.cpp -lraylib -lGL -lm

# Ejecutar
./practicador
Descargar C++ Version

Para uso móvil con interfiz táctil optimizada:

# Instalar en Termux
pkg install python clang
pip install colorama
python practicador_movil.py
