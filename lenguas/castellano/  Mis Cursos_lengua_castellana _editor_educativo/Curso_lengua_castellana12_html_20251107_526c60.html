Instalación y Configuración

Requisitos del Sistema

  • Python 3.7+ (recomendado 3.9 para compatibilidad)
  • 1GB RAM mínimo (2GB recomendado)
  • 500MB de espacio en disco
  • Conexión a Internet para instalación de paquetes

  1. Descargar Python desde python.org
  2. Abrir PowerShell como administrador
  3. Ejecutar: pip install linguistic-analysis
  4. Verificar: python -c "import linguistic; print('Instalación exitosa')"

  1. Instalar Homebrew: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  2. Instalar Python: brew install python@3.9
  3. Instalar paquetes: pip3 install linguistic-analysis

  1. Instalar Termux desde F-Droid
  2. Ejecutar: pkg update && pkg upgrade
  3. Instalar Python: pkg install python
  4. Instalar paquetes: pip install linguistic-analysis

Preguntas Frecuentes

Las herramientas están diseñadas para usuarios con conocimientos básicos de lingüística. No se requiere experiencia previa en programación para usar las interfaces web y móviles.

Sí, tenemos una aplicación móvil desarrollada con Kivy que funciona en Android e iOS. También puedes acceder a las versiones web desde cualquier dispositivo con navegador.

Las herramientas proporcionan análisis automáticos basados en reglas lingüísticas. Si bien son útiles para aprendizaje y exploración, pueden no ser perfectamente precisos en todos los casos, especialmente con textos complejos o ambiguos.

Ejemplos - Nivel Semántico-Categorial

Este nivel estudia el significado de palabras y categorías gramaticales:

  • Sustantivos: Nombran seres, objetos o conceptos (ej: "casa", "libertad")
  • Verbos: Expresan acciones, estados o procesos (ej: "correr", "existir")
  • Adjetivos: Califican o determinan sustantivos (ej: "alto", "brillante")
Texto: "El veloz zorro marrón salta sobre el perro perezoso"

Análisis:
- Sustantivos: zorro, perro
- Verbos: salta
- Adjetivos: veloz, marrón, perezoso
Ejemplos - Nivel Morfológico

Este nivel analiza la estructura interna de las palabras:

  • Lexemas: Parte de la palabra que contiene el significado principal
  • Morfemas: Unidades mínimas de significado (prefijos, sufijos)
  • Derivación: Proceso de formar nuevas palabras añadiendo afijos
Palabra: "incalculable"

Análisis:
- Lexema: "calcul"
- Prefijo: "in-" (negación)
- Sufijo: "-able" (capacidad)

Derivación: calcular → calculable → incalculable
