1. Contador de Vocales

Escribe una función que cuente cuántas vocales hay en una frase en español.

  • Código
  • Solución
// Escribe tu código aquí function contarVocales(frase) { return 0; }
// Solución en JavaScript function contarVocales(frase) { const vocales = "aeiouáéíóú"; return frase.toLowerCase().split('').filter(letra => vocales.includes(letra)).length; }

2. Verificador de Palíndromos

Escribe una función que verifique si una palabra es un palíndromo.

function esPalindromo(palabra) { // Escribe tu código aquí return false; }
function esPalindromo(palabra) { const textoLimpio = palabra.toLowerCase().replace(/\s/g, ''); return textoLimpio === textoLimpio.split('').reverse().join(''); }
