¿Qué son las palabras compuestas?

La composición es un proceso morfológico mediante el cual dos o más palabras se combinan para formar una nueva palabra con significado propio.

Definición técnica

Se llama compuesto a aquella forma que contiene más de una raíz en su interior, como lava + ropas → lavarropas o verde + blanco → verdiblanco.

Características principales:
  • Contienen dos o más raíces léxicas
  • Forman una unidad semántica y gramatical
  • Pueden escribirse como una sola palabra o con guion
  • Siguen patrones específicos de formación

Tipos de palabras compuestas

Compuestos univerbales

Se integran en una única palabra ortográfica con un solo acento principal.

Ejemplos: agridulce, sordomudo, sacapuntas

Compuestos sintagmáticos

Mantienen independencia gráfica y acentual, a veces con guion.

Ejemplos: árabe-israelí, cabeza rapada, coche bomba

Compuestos neoclásicos

Formados con raíces griegas o latinas.

Ejemplos: geografía, xenófobo, fotografía

Compuestos coordinativos

Manifiestan una relación de igualdad entre sus componentes.

Ejemplos: agridulce, claroscuro, angloamericano

Compuestos subordinativos

Establecen una relación de subordinación entre un núcleo y su complemento.

Ejemplos: bocacalle, maldecir, matamoscas

Ejemplos de palabras compuestas

Sacacorchos

Estructura: V + N (saca + corchos)

Significado: Instrumento para extraer corchos

Lavaplatos

Estructura: V + N (lava + platos)

Significado: Aparato o persona que lava platos

Parabrisas

Estructura: V + N (para + brisas)

Significado: Cristal frontal de un vehículo

Telaraña

Estructura: N + N (tela + araña)

Significado: Red tejida por una araña

Bocacalle

Estructura: N + N (boca + calle)

Significado: Entrada o comienzo de una calle

Aguamarina

Estructura: N + N (agua + marina)

Significado: Piedra preciosa de color azul verdoso

Agridulce

Estructura: A + A (agrio + dulce)

Significado: Que combina sabor agrio y dulce

Sordomudo

Estructura: A + A (sordo + mudo)

Significado: Persona que no puede oír ni hablar

Blanquiazul

Estructura: A + A (blanco + azul)

Significado: Que combina los colores blanco y azul

Geografía

Estructura: Geo- + -grafía

Significado: Ciencia que estudia la Tierra

Xenófobo

Estructura: Xeno- + -fobo

Significado: Persona que siente aversión por extranjeros

Fotografía

Estructura: Foto- + -grafía

Significado: Técnica de capturar imágenes con luz
