Análisis de Frecuencia de Palabras

Este programa analiza la frecuencia de palabras en un texto. Puedes modificarlo para analizar diferentes textos o agregar funcionalidades adicionales.

from collections import Counter
import re

def analizar_frecuencia(texto):
    # Limpiar y tokenizar el texto
    palabras = re.findall(r'\b\w+\b', texto.lower())
    # Contar frecuencia de palabras
    frecuencia = Counter(palabras)
    return frecuencia

# Texto de ejemplo
texto_ejemplo = "En un lugar de la Mancha, de cuyo nombre no quiero acordarme, no ha mucho tiempo que vivía un hidalgo de los de lanza en astillero, adarga antigua, rocín flaco y galgo corredor."

# Ejecutar análisis
resultado = analizar_frecuencia(texto_ejemplo)

# Mostrar resultados
print("Análisis de frecuencia de palabras:")
for palabra, count in resultado.most_common(10):
    print(f"{palabra}: {count}")

Práctica:

Modifica el código para:

  • Excluir palabras comunes (artículos, preposiciones)
  • Analizar un texto en inglés
  • Mostrar el porcentaje de frecuencia de cada palabra

Análisis de Cambios Fonéticos

Este código simula el análisis de características fonéticas de audio. En un entorno real, necesitarías bibliotecas adicionales como librosa.

# Simulación de análisis de características fonéticas
# En un caso real, usarías librosa para procesar audio real

def analizar_caracteristicas(audio_file):
    # En una implementación real, aquí cargaríamos el archivo de audio
    # y extraeríamos características como MFCCs
    print(f"Analizando características fonéticas de: {audio_file}")
    
    # Simulación de resultados
    caracteristicas = {
        'tono_promedio': 220.5,
        'intensidad_media': 0.75,
        'timbre': [0.2, 0.5, 0.8, 0.3, 0.6]
    }
    
    return caracteristicas

# Ejemplo de uso
resultados = analizar_caracteristicas("muestra_español.wav")

print("Características fonéticas analizadas:")
for clave, valor in resultados.items():
    print(f"{clave}: {valor}")

Detección de Préstamos Lingüísticos

Este programa detecta palabras que son préstamos entre el español y el inglés.

#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main() {
    vector<string> palabras_espanol = {"marketing", "software", "chat", "hardware", "blog"};
    vector<string> palabras_ingles = {"fiesta", "siesta", "plaza", "burrito", "taco"};
    
    cout << "Préstamos del inglés al español:" << endl;
    for (const auto& palabra : palabras_espanol) {
        cout << "- " << palabra << endl;
    }
    
    cout << "\nPréstamos del español al inglés:" << endl;
    for (const auto& palabra : palabras_ingles) {
        cout << "- " << palabra << endl;
    }
    
    return 0;
}

Comparación de Estructuras Gramaticales

Este programa compara estructuras gramaticales entre el español y el inglés.

#include <iostream>
#include <string>

using namespace std;

void analizar_estructura(const string& oracion) {
    if (oracion.find("¿") != string::npos) {
        cout << "Estructura de pregunta en español detectada." << endl;
    } else if (oracion.find("?") != string::npos && oracion.find("¿") == string::npos) {
        cout << "Estructura de pregunta en inglés detectada." << endl;
    }
    
    if (oracion.find("Do ") == 0 || oracion.find("Does ") == 0) {
        cout << "Uso de auxiliar 'do/does' típico del inglés." << endl;
    }
}

int main() {
    string oracion_es = "¿Te gusta el café?";
    string oracion_en = "Do you like coffee?";
    
    cout << "Análisis de: \"" << oracion_es << "\"" << endl;
    analizar_estructura(oracion_es);
    
    cout << "\nAnálisis de: \"" << oracion_en << "\"" << endl;
    analizar_estructura(oracion_en);
    
    return 0;
}

Registro de Neologismos

Este programa simula una aplicación para registrar nuevos términos lingüísticos (neologismos).

fun main() {
    val neologismos = mutableListOf<String>()
    
    println("Sistema de Registro de Neologismos")
    println("==================================")
    
    // Simulación de entrada de datos
    val nuevosTerminos = listOf("postureo", "cringe", "finde", "wasapear", "tuitear")
    
    for (termino in nuevosTerminos) {
        neologismos.add(termino)
        println("Neologismo registrado: $termino")
    }
    
    println("\nTotal de neologismos registrados: ${neologismos.size}")
    println("Lista completa:")
    neologismos.forEachIndexed { index, termino ->
        println("${index + 1}. $termino")
    }
}

Visualización de Datos Lingüísticos

Este código simula la creación de gráficos para visualizar datos lingüísticos.

// Simulación de visualización de datos lingüísticos
data class DatosLinguisticos(val idioma: String, val porcentaje: Int)

fun main() {
    val datos = listOf(
        DatosLinguisticos("Español", 60),
        DatosLinguisticos("Inglés", 30),
        DatosLinguisticos("Espanglish", 10)
    )
    
    println("Visualización de Distribución Lingüística")
    println("=========================================")
    
    for (dato in datos) {
        val barras = "█".repeat(dato.porcentaje / 5)
        println("${dato.idioma}: ${barras} ${dato.porcentaje}%")
    }
    
    println("\nResumen:")
    println("Total de hablantes representados: ${datos.sumBy { it.porcentaje }}%")
}
