> **Colección de mini lecciones interactivas en HTML para aprender sintaxis de lengua castellana, inglés y latín.**

## 🌟 Descripción

Este repositorio contiene una colección de **recursos educativos interactivos** diseñados para facilitar el aprendizaje de:

- 📖 **Lengua Castellana** - Sintaxis, ortografía, redacción
- 🇬🇧 **Inglés** - Gramática, vocabulario, estructuras
- 🏛️ **Latín** - Declinaciones, vocabulario básico
